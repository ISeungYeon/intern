### 이미지 분류에 knn이 좋지 않은 이유
* 컴퓨터는 이미지를 숫자로 받아들인다. 같은 클래스 안에서도 다양한 종류를 가지고 있기 때문에 단순한 차이로 이미지 분류를 하는 것은 좋지 않다. 또한 고차원 데이터는 같은 클래스 안에서도 이미지 차이가 커진다. 따라서 고차원 데이터 이미지 분류는 다루기 쉽지 않다.
